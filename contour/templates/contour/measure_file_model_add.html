Required file structure

Example showing the required file structure if two environmental variables are used. The first row represents the header. It is interpreted as "variable name 1", "variable symbol 1", "variable name 2" and "variable symbol 2". All following rows contain the numerical values of the variables, each row represents one time step. The first column is associated with variable 1, the second columne is associated with variable 2. All values must be seperated by semicolons.
